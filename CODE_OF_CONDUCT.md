# Code of Conduct

## Our standard

This project expects respectful, direct, and constructive collaboration.

Examples of expected behavior:

- giving actionable technical feedback
- assuming good intent while challenging weak reasoning
- focusing criticism on code and decisions, not people
- welcoming contributors with different backgrounds and experience levels

Examples of unacceptable behavior:

- harassment or discriminatory language
- personal attacks
- bad-faith disruption of review or discussion
- publishing private information without permission

## Enforcement

Project maintainers may remove comments, reject contributions, or restrict participation for behavior that violates this code of conduct.
